# Demo Guide

Use this folder for screenshots or a short video when submitting the project.

Recommended screenshots:

1. `./build.sh` completing successfully.
2. `./run.sh` generating reports.
3. `reports/benchmark_report.md` showing compiler rankings.
4. One generated mutated file under `reports/cases/...`.
5. A missing compiler case, if one of Clang/Flang is not installed.

The benchmark already records missing compilers in the report, so a missing-tool screenshot is acceptable as a failure-case demo.
